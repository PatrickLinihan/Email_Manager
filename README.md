# Email_Manager
Write and delete gmail emails from terminal

In order to send and delete emails from gmail you must create and app password from your google account.
The steps are as follow:
1) Go to your google account -> myaccount.google.com
2) Navigate to security
3) Under "Signing in to Google" select "app passwords"
4) create an app password -> don't have to use the selections they have just use other
5) copy that password and go to terminal
6) type $ <whatever editor you use> .bash_profile
7) in the file type "export EMAIL_PASSWORD="the password you copied form step 5""
8) save the file
9) run the script!
